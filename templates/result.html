Hasil Model Antrian M/M/2

Input
  • Laju Kedatangan (λ): {{ arrival_rate }} pelanggan/jam
  • Laju Pelayanan (μ): {{ service_rate }} pelanggan/jam
  • Jumlah Server (s): 2
Hasil Perhitungan
  • Intensitas Trafik (ρ): {{ utilization }}
  • Probabilitas Tidak Ada Pelanggan (P0): {{ prob_zero }}
  • Waktu Rata-rata dalam Sistem (W): {{ avg_time_in_system }} menit
  • Waktu Rata-rata dalam Antrian (Wq): {{ avg_time_in_queue }} menit
Rumus dan Langkah Perhitungan
1. Intensitas Trafik (ρ):

Rumus: ρ = λ / (s × μ)
Perhitungan: {{ arrival_rate }} / (2 × {{ service_rate }}) = {{ utilization }}

2. Probabilitas Tidak Ada Pelanggan (P0):

Rumus: P0 = 1 / [∑ (ρn / n!) + ∑ (ρn / (s! × sn-s))]
Perhitungan: {{ prob_zero_formula }} = {{ prob_zero }}

3. Waktu Rata-rata dalam Sistem (W):

Rumus: W = 1 / (μ - λ)
Perhitungan: 1 / ({{ service_rate }} - {{ arrival_rate }}) = {{ avg_time_in_system }}

4. Waktu Rata-rata dalam Antrian (Wq):

Rumus: Wq = W - (1 / μ)
Perhitungan: {{ avg_time_in_system }} - (1 / {{ service_rate }}) = {{ avg_time_in_queue }}
